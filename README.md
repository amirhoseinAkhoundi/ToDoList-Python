# Python ToDo List

A simple command-line ToDo List application written in Python using Object-Oriented Programming.

## Features

- Add tasks
- View tasks
- Edit tasks
- Remove tasks
- Mark tasks as completed
- Sort tasks by priority
- Show task statistics
- Save and load tasks using CSV

## Project Structure

- `main.py` - User interface
- `todo.py` - ToDo list logic
- `task.py` - Task class

## Technologies

- Python
- Object-Oriented Programming (OOP)
- CSV
